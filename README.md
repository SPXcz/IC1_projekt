Docker http server
---------------

```
$ cd http-server   
# docker build -t apache-server .
# docker run -dit --name http-server -p 80:80 apache-server
``` 
Note: The dot in the second command belongs to the command!

```
$ chmod +x downgradesudo.sh
# ./downgradesudo.sh
```
